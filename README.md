 TC_011: User Signup — Positive Test
 Description: Verify that a new user can successfully register an account.
 Steps:
1. Navigate to https://automationexercise.com/login
2. Enter Name and Email in "New User Signup!" section.
3. Click "Signup".
4. Fill in account details (Title, Password, Date of Birth).
5. Fill in address information (First name, Last name, Address, Country, State, City, Zipcode, Mobile Number).
6. Click "Create Account".
 Expected Result: Account is created, and "ACCOUNT CREATED!" message is displayed.
 Actual Result: Account created successfully. Redirection to success page confirmed.
